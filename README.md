# Todo Console Application

A robust, spec-driven command-line task manager built with Python 3.13 and the Agentic Dev Stack.

## Features
- ✅ **Add Tasks**: Capture new todos with titles and descriptions.
- ✅ **View Tasks**: See your list clearly formatted.
- ✅ **Update Tasks**: Modify details as needed.
- ✅ **Delete Tasks**: Remove old items.
- ✅ **Track Status**: Mark tasks as Complete/Incomplete.

## Setup & Usage

### Prerequisites
- Python 3.13+
- `uv` package manager

### Installation
```bash
git clone <repo-url>
cd <repo-dir>
uv sync
```

### Running the App
```bash
uv run python -m src.main
```
