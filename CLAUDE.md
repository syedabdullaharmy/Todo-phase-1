# Claude Code Instructions

## Development Workflow
1. Read `CONSTITUTION.md` for principles.
2. Check `specs_history/` for requirements.
3. Follow `TASKS.md` for progress.

## Commands
- Run App: `uv run python -m src.main`
- Run Tests: `uv run python -m unittest discover tests`

## Architecture
- **Models**: Data structures (`src/models/`).
- **Services**: Business logic (`src/services/`).
- **UI**: CLI interaction (`src/ui/`).
- **Utils**: Helpers (`src/utils/`).
